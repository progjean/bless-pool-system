# Instruções para Ícones do PWA

Para completar a configuração do PWA, você precisa adicionar os ícones na pasta `public/`:

## Ícones Necessários

1. **icon-192.png** - Ícone 192x192 pixels
2. **icon-512.png** - Ícone 512x512 pixels

## Como Criar os Ícones

Você pode usar qualquer editor de imagens ou ferramenta online para criar os ícones:

1. Crie imagens quadradas (192x192 e 512x512 pixels)
2. Use o tema do sistema (cores escuras: #1a1a2e, #16213e)
3. Salve como PNG na pasta `public/`
4. Nomeie exatamente como: `icon-192.png` e `icon-512.png`

## Ferramentas Recomendadas

- [PWA Asset Generator](https://github.com/elegantapp/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- Qualquer editor de imagens (Photoshop, GIMP, Canva, etc.)

## Nota

O sistema funcionará sem os ícones, mas para uma experiência completa de PWA, é recomendado adicioná-los.
